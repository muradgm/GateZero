# Brand Designer References

## GateZero Product Truth

- GateZero is a personal trading research, risk-control, and execution-support system.
- Core wedge: No trade without evidence. No execution without risk approval.
- Avoid profit-promise branding and AI-trading hype.

## Logo Craft Principles

Use these principles:

- silhouette clarity
- optical balance
- reduction testing
- monochrome-first validation
- small-size testing
- negative-space discipline
- geometric consistency
- type/mark relationship
- scalable vector production

## App Icon Testing

Test at:

- 16px favicon
- 24px sidebar icon
- 32px app/browser icon
- 64px product navigation
- 128px launcher
- 512px preview

## Production Requirements

The final system should include:

- Figma master components
- SVG exports
- PNG exports
- monochrome exports
- reversed exports
- color variables
- spacing rules
- clear space
- minimum sizes
- incorrect usage examples

## Avoid

- guaranteed-profit implications
- green-arrow-only symbolism
- casino/gambling visual language
- aggressive mascot treatment
- crypto-hype gradients
- fake AI magic motifs
- cluttered chart/candle clichés
