# TRD-698 Intake Directory Boundary

Status: accepted

## Goal

Restricted discovery to one checked-in research-case fixture directory.

## Boundary

Checked-in, local, read-only Gate 2 evidence inspection only. No authoring, upload, external
storage, account, credential, execution, automation, prediction, claim, or gate change.

## Acceptance

Directory, diagnostics, CLI, generated-data, UI, and relevant repository checks pass.
