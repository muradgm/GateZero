# TRD-637 Simulator Workspace Authorization

Status: accepted

## Goal

Authorize a read-only local simulator evidence workspace generated from the verified reducer.

## Allowed

Local display adapters, generated fixture data, semantic evidence panels, responsive layout, and
accessibility checks.

## Blocked

Forms, simulation controls, account connectors, credentials, broker routes, execution, automation,
prediction, claims, and gate promotion.

## Acceptance

PM, Risk, QA/Security, UI/UX, and Orchestrator accept the bounded display lane.
