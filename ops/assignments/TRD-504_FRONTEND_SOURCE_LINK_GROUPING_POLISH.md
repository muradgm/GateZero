# TRD-504 Frontend Source-Link Grouping Polish

Status: accepted

## Goal

Improve source-link scanning without adding broader UI behavior.

## Scope

- Add lightweight source-link group counts.
- Preserve local artifact references.
- Keep the source-link panel read-only.

## Blocked Scope

- External publishing, remote document browsing, account links, credential links, and execution
  links.

## Required Outputs

- Source-link grouping polish.
- Render contract guard update.
- QA_SECURITY review.
- RISK review.
- ORCHESTRATOR acceptance.

## Acceptance

Accepted when source links are easier to scan and the render guard protects the grouping affordance.
