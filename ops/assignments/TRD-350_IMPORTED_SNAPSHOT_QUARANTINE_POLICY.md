# TRD-350 Imported Snapshot Quarantine Policy

## Goal

Draft the blocked state for imported snapshots before any adapter exists.

## Scope

- Define quarantine expectations for future imported data.
- Require validation before evidence use.
- Keep current data synthetic.

## Blocked

- No import implementation.
- No provider fetch.
- No evidence usability for unvalidated imports.

## Acceptance

- Quarantine policy doc exists.
- Reviews confirm imported data remains future-phase.
