# TRD-513 Operator Workflow Evidence Card Plan

Status: accepted

## Goal

Plan an operator workflow evidence card that explains manual review state.

## Scope

- Represent manual steps as evidence, risk, decision, and learning context.
- Keep operator decision authority explicit.
- Avoid command-like controls.

## Blocked Scope

- Submit controls, approval buttons, readiness labels, execution shortcuts, and automated decisions.

## Acceptance

Accepted when the card plan supports manual review without action semantics.
