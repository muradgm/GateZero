# TRD-595 Signal Candidate Contract

Status: accepted

## Goal

Implement an evidence-only signal candidate contract for future scenario review.

## Scope

- Linked research case.
- Market-intelligence input refs.
- Evidence refs.
- Scenario action category.
- Confidence level.
- Risk review ref when needed.
- Red flags and invalidation conditions.
- Operator decision required.

## Blocked Scope

- Final recommendations, action routes, autonomous action, account access, execution paths,
  readiness labels, approval labels, certainty claims, and performance claims.

## Acceptance

Accepted when signal candidates remain evidence-only, non-final, operator-reviewed, and unable to
create an action route.
