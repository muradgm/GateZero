# TRD-542 Evidence Detail Local Data Adapter Update

Status: accepted

## Goal

Add local simulation evidence detail data to the Command Center data contract.

## Scope

- Add a static local detail record.
- Include local record identifiers, references, boundary checks, source artifacts, reproducibility
  notes, and limitation notes.

## Blocked Scope

- External requests, credential handling, account identifiers, execution paths, and action controls.

## Acceptance

Accepted when the local data is rendered and covered by frontend tests.
