# TRD-512 Simulation Evidence Detail Contract Plan

Status: accepted

## Goal

Plan a read-only contract for simulation evidence detail records.

## Scope

- Define local artifact summaries as future display inputs.
- Keep source links, run metadata, assumptions, and blockers distinct.
- Avoid action controls or trading claims.

## Blocked Scope

- External account data, broker routes, credential fields, live orders, autonomous actions, and
  prediction prompts.

## Acceptance

Accepted when the contract plan stays local, deterministic, and claim-safe.
