# TRD-347 Provider License Review Checklist

## Goal

Draft provider license review fields for future data-provider evaluation.

## Scope

- Define license, permitted use, attribution, redistribution, and retention review items.
- Keep provider use blocked until reviewed.
- Avoid provider-specific integration.

## Blocked

- No provider account setup.
- No API keys.
- No real market data ingestion.

## Acceptance

- License checklist doc exists.
- Reviews confirm no provider access was added.
