# TRD-517 Source-Link Density Follow-Up Review

Status: accepted

## Goal

Review source-link density after the frontend source-link count polish.

## Scope

- Decide whether source links need future grouping, filtering, or summaries.
- Keep all links local.
- Avoid turning docs into a public publishing surface.

## Blocked Scope

- External publishing, remote document sync, account-linked documents, and marketing claims.

## Acceptance

Accepted when density pressure is documented without adding UI breadth.
