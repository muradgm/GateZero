# TRD-343 Adapter Authorization Blocker Inventory

## Goal

Inventory every missing authorization before real historical data adapter work can begin.

## Scope

- List required risk, QA, provenance, license, and security blockers.
- Keep all adapter work future-phase.
- Preserve synthetic fixtures as the only current data source.

## Blocked

- No provider integration.
- No network fetch path.
- No API keys, tokens, broker credentials, or account identifiers.

## Acceptance

- Source-linked blocker inventory exists.
- Reviews confirm adapter implementation remains blocked.
