# TRD-324 Blocked Evidence Docs Coverage Recheck

## Goal

Recheck that blocked-evidence documents are indexed, source-linked, and visible to local guards.

## Scope

- Review missing-candle, stale-data, duplicate-signal, parameter-immutability, and summary docs.
- Add a coverage recheck record.
- Preserve Gate 1 historical-backtesting-only scope.

## Blocked

- No claims that blocked evidence is usable.
- No loosening of blocker language.

## Acceptance

- Coverage doc is indexed and guard-visible.
- Review records confirm the blocked-evidence chain remains intact.
