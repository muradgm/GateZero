# TRD-546 Evidence Detail Source-Link Rendering

Status: accepted

## Goal

Render local source-link and source-artifact references in the evidence detail panel.

## Scope

- Display local source-link map references.
- Display local `docs/` and `ops/` source artifacts.
- Keep references as text-only local artifacts.

## Blocked Scope

- External publishing, account URLs, credential links, or unreviewed source expansion.

## Acceptance

Accepted when source references are visible in tests and browser QA.
