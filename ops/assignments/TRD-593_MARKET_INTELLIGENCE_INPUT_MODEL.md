# TRD-593 Market Intelligence Input Model

Status: accepted

## Goal

Implement the source-linked market-intelligence input model for future scenario analysis.

## Scope

- Market condition, news/event, red-flag, and signal-candidate input categories.
- Source refs, confidence level, red flags, invalidation conditions, risk review requirement, and
  operator decision requirement.

## Blocked Scope

- Final recommendations, automated decisions, account routes, execution paths, certainty claims,
  profit claims, and risk-gate loosening.

## Acceptance

Accepted when market-intelligence inputs remain sourced, non-final, risk-reviewed, and
operator-required.
