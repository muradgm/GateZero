# TRD-523 Operator Workflow Evidence Fixture Plan

Status: accepted

## Goal

Plan fixtures for manual operator workflow evidence.

## Scope

- Manual review state.
- Evidence reference.
- Risk note reference.
- Learning-event reference.

## Blocked Scope

- Automated decisions, submit controls, readiness labels, approval labels, and execution shortcuts.

## Acceptance

Accepted when fixtures are bounded to manual workflow evidence only.
