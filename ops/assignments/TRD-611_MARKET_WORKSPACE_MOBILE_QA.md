# TRD-611 Market Workspace Mobile QA

Status: accepted

## Goal

Keep dense scenario evidence readable on small screens.

## Scope

- Verify single-column layout, bounded spacing, mobile table labels, and long-reference wrapping.
- Inspect the local workspace at a narrow viewport.

## Blocked Scope

- Mobile application work, action controls, external routes, and output channels.

## Acceptance

Accepted when scenario panels remain readable without overlap or horizontal page overflow.
