# TRD-547 Evidence Detail Mobile Visual QA

Status: accepted

## Goal

Verify the evidence detail panel remains readable on small screens.

## Scope

- Use responsive grid collapse.
- Preserve readable cards, source references, and adjacency blocks.

## Blocked Scope

- Dense layouts that overflow mobile viewports or hide risk/limitation context.

## Acceptance

Accepted when browser QA confirms the panel renders on the local preview.
