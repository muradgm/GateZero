# TRD-538 Evidence Source-Link Map Implementation

Status: accepted

## Goal

Represent source-link map references inside the simulation evidence detail fixture.

## Scope

- Add source-link map references.
- Require at least one source-link reference.
- Keep the references local and documentation-backed.

## Blocked Scope

- External publishing, broker account links, credential links, or unreviewed source expansion.

## Acceptance

Accepted when source-link map references are required by tests and indexed in docs.
