# TRD-765 Scenario Invalidation Evaluation

Status: accepted

## Goal

Evaluate scenario invalidation criteria against frozen local evidence only on explicit operator
request.

## Output

- Invalidation evaluation contract, evaluator, checks, fixture, and workspace summary.

## Acceptance

- Scheduled evaluation is forbidden.
- Stale evidence requires revision; missing evidence blocks evaluation.
- No notification, alert, or automatic action exists.
