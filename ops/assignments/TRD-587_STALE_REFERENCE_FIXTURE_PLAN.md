# TRD-587 Evidence-Control Stale-Reference Fixture Plan

Status: accepted

## Goal

Plan stale-reference negative cases for workspace evidence, artifact inventory, and event records.

## Scope

- Local stale-source fixtures.
- Nonlocal reference rejection.
- Missing-source and blocked-freshness cases.

## Blocked Scope

- External polling, cloud sync, remote ingestion, account data, execution records, and action
  routing.

## Acceptance

Accepted when stale-reference cases are represented in negative contract tests and remain local.
