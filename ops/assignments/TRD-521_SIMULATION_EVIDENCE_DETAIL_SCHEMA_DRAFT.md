# TRD-521 Simulation Evidence Detail Schema Draft

Status: accepted

## Goal

Draft the local read-only simulation evidence detail record shape before implementation.

## Scope

- Local evidence metadata.
- Source artifact references.
- Assumption and blocker references.
- Reproducibility notes.

## Blocked Scope

- Credentials, account connectivity, live execution, autonomous action, AI prediction, approval
  semantics, readiness semantics, and performance claims.

## Acceptance

Accepted when the schema draft is documented as planning-only and ready for later guarded
implementation.
