# TRD-586 Gate 2 Note Model Source-Link Plan

Status: accepted

## Goal

Define how manual operator notes link back to local evidence sources for the Strategy Review
Workspace.

## Scope

- Manual local operator notes only.
- Required links to research case, evidence detail, artifact ids, and source refs.
- Limitation and redaction fields remain visible beside note content.

## Blocked Scope

- Automated decisions, action controls, account access, credentials, external storage, output
  channels, approval claims, performance claims, and risk-gate loosening.

## Acceptance

Accepted when the note model is planned as manual, local, source-linked, and useful for `TRD-592`.
