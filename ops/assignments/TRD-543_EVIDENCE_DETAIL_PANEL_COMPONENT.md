# TRD-543 Evidence Detail Panel Component

Status: accepted

## Goal

Render the evidence detail panel inside the existing Evidence section.

## Scope

- Add detail cards for core records, boundary checks, workflow references, risk references, artifact
  summaries, failure-mode evidence, source-link maps, and local source artifacts.
- Use semantic headings, lists, and definition lists.

## Blocked Scope

- Buttons, forms, dispatch controls, account connection controls, or decision automation.

## Acceptance

Accepted when tests verify the panel renders without action controls.
