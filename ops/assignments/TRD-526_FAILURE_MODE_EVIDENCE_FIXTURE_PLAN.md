# TRD-526 Failure-Mode Evidence Fixture Plan

Status: accepted

## Goal

Plan fixtures for failure-mode evidence states.

## Scope

- Stale input state.
- Missing artifact state.
- Invalid assumption state.
- Schema mismatch state.
- Replay mismatch state.

## Blocked Scope

- Auto-remediation, execution routing, credential workflows, and action dispatch.

## Acceptance

Accepted when fixture planning surfaces blockers without automated resolution.
