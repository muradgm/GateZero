# TRD-527 Simulation Evidence Source-Link Map Plan

Status: accepted

## Goal

Plan a local source-link map for future simulation evidence records.

## Scope

- Deduplicated local paths.
- Assignment and operation references.
- Contract and fixture references.
- Guard references.

## Blocked Scope

- External publishing, remote sync, account-linked sources, credential paths, and marketing claims.

## Acceptance

Accepted when source-link planning remains local and deduplicated.
