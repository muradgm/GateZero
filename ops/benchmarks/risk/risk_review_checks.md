# Risk Benchmark — Strategy Review

Risk review must answer:

- What is the maximum expected loss condition?
- What invalidates the strategy?
- What happens after consecutive losses?
- How is position size calculated?
- What stops trading for the day?
- Can the strategy trade stale data?
- Can duplicate signals create duplicate orders?
- Can a failed broker call leave state inconsistent?
- Can the operator override controls?

Any unanswered critical question blocks promotion.
