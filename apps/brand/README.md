# Trader Frame — Brand Identity System

A runnable React + TypeScript + Vite brand showcase for Trader Frame.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown by Vite, usually `http://localhost:5173`.

## Production build

```bash
npm run build
npm run preview
```

## Notes

- The supplied Trader Frame logo is included in `src/assets`.
- Fonts are loaded from Google Fonts.
- The primary wordmark is stored as an outlined SVG in `src/assets/trader-frame-logo.svg`.
- The Gate Frame symbol is stored as a standalone SVG in `src/assets/trader-frame-gate-frame.svg`;
  its micro drawing is intended for 16–24px use.
- The identity direction is conceptual and does not constitute trademark clearance.
