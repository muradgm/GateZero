# GateZero Brand Identity System

Reference board: `docs/brand_handoff/reference/gatezero-identity-board.png`

## Brand Essence

The control layer before every trade.

## Brand Definition

GateZero is a personal trading research, risk-control, and execution-support system.

No trade without evidence. No execution without risk approval.

## Core Message

Evidence in. Risk approved. Execution controlled.

## Logo Direction

The GateZero logo uses an angular dimensional `G` mark with an inner gate and keyhole.

The mark represents:

- evidence entering the system
- controlled access before execution
- risk approval as a gate
- operator accountability
- disciplined action

## Logo System

Required variants:

- primary logo
- horizontal lockup
- stacked lockup
- symbol only
- wordmark only
- app icon
- favicon
- monochrome black
- monochrome white reversed
- on dark background
- on light background
- small-size simplified symbol

## Color Palette

| Name          | Hex       | Use                                               |
| ------------- | --------- | ------------------------------------------------- |
| Deep Purple   | `#1A1028` | Primary dark brand base                           |
| Violet        | `#7C5CFF` | Primary accent, ZERO wordmark, dimensional facets |
| Teal          | `#00C6A7` | Data, approved state, control accents             |
| Slate         | `#EAEFF5` | Light surfaces and neutral UI                     |
| Signal Orange | `#FF8A00` | Warning/signal accent only                        |

## Typography

Primary font:

- Raleway
- SemiBold / Bold / ExtraBold
- Used for logo, headings, and major brand moments

Secondary font:

- Inter
- Used for body text, UI labels, data, and product copy

## Tone Of Voice

- Calm, confident, professional.
- Facts over opinions.
- Evidence before action.
- Discipline over impulse.
- Control before execution.

## Usage Rules

Do not:

- change the logo colors
- distort the logo
- add outlines or effects
- use low-contrast versions
- alter the keyhole or gate elements
- rotate the logo
- imply guaranteed trading profit
- use casino, crypto-hype, or get-rich-quick visuals

## Minimum Size

- Horizontal lockup minimum width: 120px
- Symbol minimum width: 24px
- Small-size symbol checks: 64px, 32px, 16px

## Clear Space

Clear space around the logo is defined by `X`, where `X` is the height of the inner keyhole element.

Maintain at least `X` on all sides of the logo in layouts.
