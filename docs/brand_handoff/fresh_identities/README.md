# TraderFrame Fresh Identity Options

These are deliberately separate from the existing purple geometric TraderFrame identity family.

The goal is to test three new visual territories with different color systems, type direction,
symbol language, and emotional posture while preserving the product truth:

> Research the trade. Frame the risk. Execute only when allowed.

## Options

- `clearance-protocol`: compliance-grade control, off-white paper, red clearance stamp, technical
  mono.
- `northstar-terminal`: refined bracketed northstar, abyss, frost, north teal, brass checkpoint.
- `blackline-ledger`: austere editorial ledger, black ink, paper surface, restrained red risk mark.

## Evaluation Lens

Judge each option by:

- small-size recognizability
- seriousness and trust
- distance from trading hype
- usability in the command center
- distinctiveness from the previous purple TF marks
