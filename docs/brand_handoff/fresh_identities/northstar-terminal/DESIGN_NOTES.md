# TraderFrame Fresh Identity - Northstar Terminal

## Posture

Navigational, focused, terminal-grade.

This direction treats TraderFrame as a decision navigation system. It is less about approval stamps
and more about finding orientation in noisy markets.

## Color System

- Abyss: `#071018`
- Frost: `#DDEBE7`
- North Teal: `#1E8F86`
- Brass Checkpoint: `#C9963E`
- Haze: `#EEF3F0`

## Typography

- Display: `Aptos Display`
- UI/data: `JetBrains Mono`
- Fallback: `Segoe UI`, `Inter`, `Arial`

## Symbol

The symbol returns to the original northstar idea: a bracketed terminal frame, a precise four-point
orientation star, and a brass checkpoint. The execution is tightened so the mark reads as
"navigation under control" without becoming decorative.

## Best Use

Use this if TraderFrame should feel like a premium navigation terminal for strategy review and risk
orientation.
