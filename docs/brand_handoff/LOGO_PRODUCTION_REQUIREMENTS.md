# GateZero Logo Production Requirements

## Required File Formats

- Figma source file
- SVG
- PDF
- PNG transparent
- PNG on white
- PNG on dark navy
- favicon PNG
- app icon PNG 512x512 and 1024x1024

## Quality Tests

- Test mark at 16px, 24px, 32px
- Test monochrome on white
- Test reversed on navy
- Test favicon legibility
- Test app icon in rounded square
- Test dashboard header lockup
- Check wordmark spacing
- Check all paths for messy overlaps
- Remove raster artifacts
- Outline final wordmark in production SVGs

## Production Warning

Do not use raw AI-generated raster art as the final logo.

Generated logo images may be used as art direction only. The final mark must be rebuilt as clean
vector geometry.
