# TraderFrame Identity Alternatives

This folder contains additional identity explorations for the active TraderFrame brand.

The current active direction remains in `docs/brand_handoff/logo_exports/`.

## Directions

- `signal-window/`: abstract framed decision window with signal and approval axis.
- `risk-ledger/`: operational ledger mark with evidence columns and controlled execution rail.

## Shared Brand Rule

All directions must preserve the same product truth:

> Research the trade. Frame the risk. Execute only when allowed.

None of these identities should imply profit certainty, trading signals, casino energy, or automated
wealth creation.
