# Conviction Atlas Intelligence Language R&D

## Scope

This branch evaluates five visual intelligence languages without changing the approved production baseline.

1. Signal Rivers
2. Conviction Topography
3. Evidence Constellation
4. Decision Lenses
5. Scenario Braids

## Shared criteria

Each direction is scored from 1–10 on:

- Clarity
- Ownability
- Scalability
- Performance
- Accessibility

Maximum score: 50.

## Recommendation

Use **Conviction Topography** as the core brand metaphor because it is the closest fit to the approved final product and has the strongest ownability. Borrow interaction clarity and accessibility patterns from **Decision Lenses**. Use **Scenario Braids** selectively for transitions between competing outcomes.

The numeric ranking is evidence, not an automatic selection. The final direction must balance product distinctiveness with usability and production feasibility.

## Review gate

Do not merge this branch until:

- the five prototypes are reviewed side by side;
- one core metaphor is selected;
- borrowed behaviors from secondary directions are documented;
- mobile and reduced-motion behavior is accepted;
- the selected direction is mapped back to the approved Evidence Gate artifacts.
