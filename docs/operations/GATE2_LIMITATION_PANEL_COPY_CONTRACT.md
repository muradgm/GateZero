# Gate 2 Limitation Panel Copy Contract

## Summary

TRD-471 defines copy rules for limitation panels.

## Required Copy Posture

- Lead with "Limitation", "Assumption", "Blocked", or "Local evidence".
- Keep assumptions adjacent to evidence summaries.
- Use specific caveats instead of vague warnings.
- State that simulation evidence does not authorize trading.

## Blocked Copy

- ready, approved, safe, certified, promoted, profitable, deployable, production-ready, tradeable,
  broker-ready, or live-ready.

## Result

Future limitation panels must reduce false confidence and keep the operator inside the protected
loop.
