# Gate 2 Simulation Failure Mode Plan

TRD-438 plans failure modes for future Gate 2 mechanics.

## Failure Modes

- Missing evidence bundle.
- Missing risk review.
- Missing operator action.
- Credential attempt.
- External account attempt.
- Live route attempt.
- Autonomous transition attempt.
- Nondeterministic replay.
- Performance or readiness claim.

## Response

Failures block local simulation evidence and route to Gate 2 planning hold or Gate 1 maintenance.
