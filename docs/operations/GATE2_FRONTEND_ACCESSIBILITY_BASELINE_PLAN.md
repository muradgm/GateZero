# Gate 2 Frontend Accessibility Baseline Plan

## Summary

TRD-475 defines accessibility requirements for future read-only frontend screens.

## Baseline Requirements

- Keyboard reachable navigation and panels.
- Visible focus states.
- Semantic landmarks.
- Descriptive table captions.
- Text alternatives for status indicators.
- Sufficient contrast for risk and limitation states.
- Responsive layouts that keep evidence, risk, and limitations readable.

## Result

Accessibility must be planned as part of the control plane, not as a late polish pass.
