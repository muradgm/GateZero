# Gate 2 Negative Contract Tests

TRD-430 adds negative tests for Gate 2 contracts.

## Covered Boundaries

- Wrong gate or scope.
- External access.
- Execution path.
- Credential requirement.
- Live route.
- Automated action or transition.
- Approval or performance claims.
- Missing risk or operator evidence.
- Real account data or secrets in negative fixtures.
