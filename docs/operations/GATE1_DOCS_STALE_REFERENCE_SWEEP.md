# Gate 1 Docs Stale-Reference Sweep

TRD-378 records stale-reference classes to watch in Gate 1 documentation.

## Sweep Result

Historical records may mention older gate language when they describe past work. Active docs,
trackers, command-center data, and next-action records must use current Gate 1 wording:
`G1_BACKTESTING` and `historical_backtesting_only`.

## Watch Classes

- Active docs that say Gate 0 is current.
- Readiness language used as permission.
- Validation language used as strategy approval.
- Gate 2 references that sound authorized instead of blocked.

## Boundary

This sweep does not rewrite historical records or activate future phases.
