# Gate 1 Completion Criteria Draft

## Purpose

Draft the minimum evidence required before Gate 1 can be considered complete enough to request a
future Gate 2 planning review.

## Completion Criteria

- Backtest run assembly contract exists and passes validation.
- Metric report evidence contract exists and rejects claims.
- Reproducibility comparison fails closed.
- Assumption risks include bad-case fixtures.
- Operator decision events preserve risk review and operator authority.
- Full local verification passes.

## Non-Readiness

Meeting these criteria would not authorize paper trading, live trading, broker integration,
autonomous execution, AI prediction, strategy approval, readiness, or performance claims.
