# Gate 2 Evidence Detail Panel Component

TRD: TRD-543

## Implementation

The Evidence section renders a compact detail panel with core records, boundary checks, workflow
references, risk references, artifact references, failure-mode evidence, source-link maps, and local
source artifacts.

## Boundary

The panel uses static markup, lists, and definition lists only. It contains no buttons, forms, or
action controls.
