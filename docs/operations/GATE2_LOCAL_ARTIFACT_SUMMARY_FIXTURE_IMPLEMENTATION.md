# Gate 2 Local Artifact Summary Fixture Implementation

TRD: TRD-536

## Implementation

source artifact paths from local `ops/` or `docs/` records.

## Boundary

External URLs, account routes, credential payloads, and live data routes are rejected as source
artifacts for this contract.
