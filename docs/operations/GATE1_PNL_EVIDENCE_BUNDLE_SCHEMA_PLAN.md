# Gate 1 PnL Evidence Bundle Schema Plan

## Purpose

Plan the PnL evidence bundle shape for Gate 1 historical backtesting.

## Required Bundle Signals

- Directional PnL check identifiers.
- Included instruments.
- Cross-currency case marker.
- JPY precision case marker.
- Declared cost consistency status.
- Limitations.

## Boundary

The bundle remains schema-only and evidence-only. It is not a strategy report, approval artifact,
readiness marker, or execution artifact.
