# Gate 1 Provider License Review Checklist

Provider licensing must be reviewed before any data provider is used.

## Checklist

- Provider name and source URL.
- Permitted personal research use.
- Redistribution constraints.
- Retention constraints.
- Attribution requirements.
- API terms that affect storage or derived data.
- Review owner and review date.

## Boundary

This checklist does not authorize provider access, account setup, API key handling, or data
ingestion.
